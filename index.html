rwalgebra.js

A math library for parsing, storing, and manipulating algebraic expressions.
Supports operations in the real and complex domains.

Demo

Cartesian & Polar Forms

Output the cartesian and polar forms of the expression. NOTE: Polar form can only be computed if the expression does not contain any unknown variables (i.e. only constants).

Data Structures

      

         RWalgebra.js consists of 2 data structures for algebraic manipulation:
      

      
    
         
  • Expression
    • 
         
  • Equation
    • 
      

      

         The Expression object is used for storing and manipulating 
         algebraic expressions (i.e. "x + 5j") as shown in the Demo. 

         The  Equation object is used to store algebraic equations
         of the form x + y = 5j - z.
